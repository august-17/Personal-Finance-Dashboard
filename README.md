# Personal Finance Dashboard

A Python-based desktop application for managing personal finances through an intuitive graphical interface.

The application allows users to record income and expenses, categorize transactions, track financial activity, analyze spending patterns, and visualize financial data through interactive charts and reports.

## Features (Planned)

- Add income and expense transactions
- Categorize financial records
- Store transaction data in CSV format
- View transaction history
- Calculate total income, expenses, and balance
- Search and filter transactions
- Generate financial summaries
- Visualize spending patterns with charts
- User-friendly graphical interface using Tkinter

## Technologies

- Python
- Tkinter
- CSV
- Pandas
- Matplotlib
